# JSON Tree Visualizer (Upgraded)
Features:
- JSON input and validation
- Tree visualization using React Flow
- Search by JSONPath (e.g., $.user.address.city, $.items[0].name)
- Highlight matching nodes and pan/zoom to matched node
- Dark / Light theme toggle (Theme B - modern dark)
- Modular components: App, JsonInput, JsonTree, SearchBar
How to run:
1. Ensure Node.js >= 14 is installed.
2. In project root: `npm install`
3. `npm start`
Files included: src/App.js, src/index.js, src/components/JsonInput.js, src/components/JsonInput.css, src/components/JsonTree.js, src/components/JsonTree.css, src/components/SearchBar.js, src/App.css, package.json
